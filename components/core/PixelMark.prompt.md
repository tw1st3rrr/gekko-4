**PixelMark / Wordmark** — the brand's signature mark, a 2×2 cluster of green squares that reads as construction "pixels". `Wordmark` pairs it with the ГЕККО name in the display face. Use in nav, footer, and any brand lockup.

```jsx
<Wordmark />
<PixelMark size={8} />
```

The same motif appears as the fading 3-dot bullet inside `SectionLabel`.
