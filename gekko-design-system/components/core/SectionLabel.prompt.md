**SectionLabel** — the small uppercase, wide-tracked eyebrow that introduces every section ("как это работает", "стоимость", "отзывы"). Led by the fading 3-square pixel motif and always in the accent green. Lowercase the text — the CSS uppercases it.

```jsx
<SectionLabel>как это работает</SectionLabel>
<SectionLabel center>стоимость</SectionLabel>
```
