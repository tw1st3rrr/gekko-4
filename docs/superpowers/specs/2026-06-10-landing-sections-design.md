### 2. Как это работает (`HowItWorksSection`)

3 шага с SVG-иллюстрациями (нарисованные inline, в стиле brick.me):

| Шаг | Иконка | Заголовок | Описание |
|-----|--------|-----------|----------|
| 1 | SVG: загрузка фото на устройство | Загрузи фото | Нажми «Начать проект» и загрузи любое фото — портрет, пейзаж, питомец, любимая сцена |
| 2 | SVG: экран с пикселями и настройками | Настрой в пикселизаторе | Выбери формат, стиль (ч/б, сепия, цвет), посмотри превью — редактируй пока не понравится |
| 3 | SVG: коробка с готовым набором | Получи набор и собери | Набор с кирпичиками, схемой и базовыми платами придёт почтой. Собери сам или подари |

Фон секции: `#faf9f5` (бежевый, как весь сайт).  
Горизонтальная сетка на десктопе, вертикальная на мобиле.

## Технические решения

- Все новые секции — отдельные функции-компоненты в `App.tsx`
- Никакого нового роутинга — всё на одной странице
- FAQ-аккордеон: локальный state `openIndex`, без внешних библиотек
- Калькулятор: локальный state `format` + `color`, без внешних зависимостей
- SVG-иллюстрации для How it Works: inline JSX, не отдельные файлы
- Изображения brick.me как заглушки — с `onError` fallback на градиент
- Цветовая схема: `#faf9f5` фон / `#3d3929` текст / `#c96442` акцент (как весь сайт)
- Чередование фонов секций: бежевый → белый → бежевый → тёмный → белый для визуального ритма
